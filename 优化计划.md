4、TF-IDF 权重计算 (信息检索、文本挖掘、搜索引擎常用算法) 
建议使用 第三方库+自定义工具包(复用性高)  
词频 TF = 某个词在文档中出现的次数 / 文档总词数
逆文档 IDF = log(语料库中文档总数 / 包含该词的文档数 + 1)
TF-IDF = TF × IDF
可集成到 对话历史over、摘要生成over、恶意检测等  

对话历史、摘要可以小幅优化
配置检测词库(恶意检测等)

针对 tf-idf 优化：
tf-idf --> bm25
文档长度归一化： tf-idf对长文档有偏见(词频更高)
             bm25通过参数b控制文档长度影响，避免过度偏向长文档
词频权重计算： tf-idf线性词频权重，高频词可能过度影响
             bm25 非线性饱和函数，词频增长到一定程度后权重增长放缓
参数可调性： k1 参数 - 控制词频饱和度（推荐值 1.2-2.0）
            b 参数 - 控制文档长度归一化程度（推荐值 0.75）
